Downloads

Argo can be downloaded in three forms:

  • the standard jar, with source code included,
  • a compact jar, with no source, and no debug information,
  • or as the full source code including tests etc. using Subversion from https://argo.svn.sourceforge.net/svnroot/argo/tags/0.10

It has no runtime dependencies, and all dependencies used in tests are included in that download.
