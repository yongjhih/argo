Introduction

Argo is a JSON parser and generator for Java. It offers three parse interfaces - a push parser, a pull parser, and a DOM syle parser. It is open source, and free for you to use.

The latest version of Argo available for download is 0.10. The javadoc is also available online.

Example

A brief example demonstrates the DOM style parser. The example is based on the following JSON, which is assumed to be available in a String called jsonText.

{ "name":"Black Lace", "sales": 110921, "singles":[ "Superman", "Agadoo" ] }

We can use Argo to get the second single like this:

JsonNode json = new JdomParser().parse(jsonText); String secondSingle = anObjectNodeWithField("singles") .with(anArrayNodeWithElement(1) .with(aStringNode())) .getValue(json);

On the first line, we parse the JSON text into an object hierarchy. We could have navigated this ourselves, but in this case, it's easier to let a JsonNodeSelector take the strain.

We build up a selector:

  1. that matches a JSON object (the root of the example) with a field called singles,
  2. we specify that singles contains an array, and we select the second element,
  3. we specify that the element should be a string,
  4. and finally, we get the value of our string

If we check the secondSingle variable, we will find, as expected, it contains the String "Agadoo".
